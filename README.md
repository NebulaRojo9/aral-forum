# AcadForum: a digital academic forum
## About
AcadForum is a website forum that serves as academic information exchange. Users can post academic-related questions that allow dissemination to range of audiences or respond to queries to satisfy one's curiosity. Any academic field is allowed. As such, people from different expertise can share different kinds of information together.

## Webpages (based from design on Figma)
**Create Account/Login Page**
- Admin user can only be created through manipulating elements in database

**Landing Page**
- see lists of classes available
- can join classes that are in public visibility setting
- otherwise, they can enter class code to join private class

**Profile View**
- username, birthdate, bio, subject strength, subject weaknesss

**Class Page**
- created by user
- A user who created this post automatically becomes creator. They can assign others to become co-manager. They can also invite others to join class code

    - Admin (1 to many): can see all classes (public or private). The supervisor and can ban or delete users.
2. Website interface

    - must not have any other unnecessary features (e.g. tic-tac-toe game) to optimally focus on its purpose of academic information exchange
    - As much as possible if it is flexible also for mobile view
